计算机概论

计算机由5个硬件单元组成：输入设备、输出设备、CPU内部的控制单元、CPU内部算术逻辑单元、内存组成
CPU分为两种一种是复杂指令集（Complex Instruction Set Computer,CISI)，目前市场上主要有AMD、inter、VIA等X86架构的CPU；另一种是精简指令集（Reduced Instruction Set Computing,RISC)主要有sun公司的SPARC系列以及IBM公司的Power Architecture系列与ARM系列等，精简指令集通常使用在手机、pad、导航系统、网络设备等
接口设备：存储设备、显示设备、网络设备
计算上常用的计算单位

进位制 K M G T P
二进制 1024 1024K 1024M 1024G 1024T
十进制 1000 1000K 1000M 1000G 1000T
主板：（1）北桥负责连接速度较快的CPU，内存与显卡等组件（2）南桥负责速度较慢的周边接口，包括硬盘、USB、网卡等；x86架构和AMD架构的主要区别在于AMD讲内存集成到CPU中，而x86架构是通过北桥连接cpu和内存。

AJAX学习笔笔记

通过AJAX技术，可以实现无刷新的方式更新HTML元素中的内容。AJAX技术是由4种技术组成的结合体，分别是javascript、CSS、DOM、XMLHttpRequest.前3种技术是客户端技术，XMLHttpRequest组件和服务端的关系密不可分，使用这个组件可以从服务器端获得数据做出更好的系统。
XMLHttpRequest的原理：它是发送一个HTTP请求的客户端组件，开发人员可以根据不同的情况选择以同步或者异步的方式来发送HTTP请求，并获得服务器端的相应消息，然后使用DOM技术找到要更新的的某些HTML元素，并用这些数据进行更新。
